Welcome to openLCA v1.4

Getting started

With the new version of openLCA it is really easy to get started. You can just import a database from an existing zolca-file and open it from the navigation on the left side. Then open the database with a simple double click.

Download data

You can get professional LCA databases ready to use in openLCA from openLCA Nexus. There you can find professional LCI databases like ecoinvent or GaBi but also free databases and other resources like LCIA methods.

Read the documentation

There is an openLCA Wiki and a forum available where you can learn more about openLCA and ask questions. Additional documentation is also available on our website.

It is open source
