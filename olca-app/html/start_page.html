Welcome to openLCA!
Version 1.4 for Windows 64 bit.

openLCA is a high performance Life Cycle Assessment (LCA) and sustainability assessment software.

Truly open source. Simply free, as in “free beer”. Created and maintained since 2006 by GreenDelta, Berlin, www.greendelta.com.

openLCA is licensed under the Mozilla Public License, 1.1 (what does this mean#?).

What's new

For the 1.4 version, we completely redesigned and “renovated” openLCA. Results are

  • a drastically improved performance of about 400% (Performance Test)
  • numerous usability improvements a more comprehensive list is available,here
  • many new features a more comprehensive list is provided here,here

Getting started

When you start openLCA the first time, it does not contain any data. With the new version of openLCA, it is really easy to get started. You can just import a database from an existing zolca-file and open it from the navigation on the left side. Then open the database with a simple double click.

This is explained in a video,here

And in the wiki,here

Searching and downloading data

A broad range of LCA databases are available in openLCA nexus, where you can search for single data sets and also for complete databases, and download them into openLCA. The databases are all fit for openLCA, several databases can be combined.

This is also explained in avideo
