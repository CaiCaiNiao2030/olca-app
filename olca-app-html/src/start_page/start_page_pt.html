O que há de novo?

Para a versão 1.4, renovamos completamente o openLCA. Os resultados são:

  • Melhor desempenho, cerca de 400%
  • Numerosas melhorias em usabilidade
  • Muitas características novas

O que há de novo na versão 1.4.1? descubra aqui.

Começando

Com a nova versão do openLCA, é realmente fácil de começar. Você pode simplesmente importar um banco de dados de um arquivo zolca existente, e abri-lo na janela de navegação com um simples clique duplo.

Algumas funções básicas do openLCA também são explicadas no vídeo.

Arquivos para download

Uma ampla gama de bancos de dados de ACV estão disponíveis no openLCA Nexus, onde você pode procurar datasets individuais e bases de dados completas e transferi-los para importação no openLCA. As bases de dados estão prontas para uso no openLCA e vários bancos de dados podem ser combinados.

Métodos AICV estão disponíveis a partir do openLCA. Faça o download.

Para saber como importar bancos de dados e métodos de avaliação de impacto no openLCA, acesse o vídeo.
